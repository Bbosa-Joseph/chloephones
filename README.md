# Chloephones

A web project for Chloephones.

## Getting Started

Clone the repository and open `index.html` in your browser to get started.

```bash
git clone https://github.com/Bbosa-Joseph/chloephones.git
cd chloephones
```

## Project Structure

```
chloephones/
├── index.html      # Main entry point
├── style.css       # Stylesheet
└── README.md       # Project documentation
```
